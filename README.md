# OCR Reader

Extract text from images using OCR technology.

## Features
- Drag & drop image upload
- Real-time text extraction
- Edit extracted text
- Copy/download results
- Dark mode support

## Quick Start
```bash
npm install
npm run dev
```

## Usage
1. Upload an image
2. Click "Extract Text"
3. Copy or download the result

## Tech Stack
- React + TypeScript
- Tesseract.js
- Tailwind CSS
- Vite
